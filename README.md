## 📁 Assignments

### Assignment #1: AI Chatbot Application

A Windows Forms application that integrates with **Google Gemini API** to provide an intelligent conversational AI assistant.

#### Features
- Windows Forms UI
- Google Gemini API integration
- Async operations for smooth UI
- Error handling
- Input validation

#### Technologies
| Technology | Version |
|------------|---------|
| C# | 9.0 |
| .NET Framework | 4.8 |
| Windows Forms | - |
| Google Gemini API | gemini-2.0-flash-exp |
| Newtonsoft.Json | 13.0.3 |

#### Setup
1. Get API key from https://aistudio.google.com/apikey
2. Open `AI_Chatbot_App/Source_Code/Form1.cs`
3. Replace `YOUR_API_KEY_HERE` with your actual API key
4. Build and run in Visual Studio
